CRIACAO DE VALORES:

Repare que todo valor instanciado vai para enviroment no R Studio!

Dica: Por mais que possamos usar o simbolo de igualdade, na atribuicao de valores, é mais elegante que usemos a setinha! O simbolo de igual é melhor deixar para funcoe, enquanto a atribuicao de variaveis é mais lógico que usemos a setinha

Ex:

testeA <- 3*2
4*8 -> testeB

Particulamente me é mais interessante a situação do testeA.

ARITIMÉTICA LÓGICA:

R encara TRUE como 1, e FALSE como 0! Essa informação é muito util para casos futuros

testeD <- TRUE

class (testeD)
[1] "logical"
testeD + testeA
[1] 7
FALSE + testeA
[1] 6
TRUE + TRUE         # = 2 
[1] 2
TRUE + FALSE        # = 1
[1] 1
TRUE + FALSE/FALSE  # NaN, not a number
[1] NaN
TRUE/FALSE          # INF = INFINITO
[1] Inf
!TRUE       # Negacao de verdade
[1] FALSE
!FALSE      # Negacao da mentira
[1] TRUE
TRUE|FALSE  # OU lógico
[1] TRUE
TRUE&FALSE  # E lógico
[1] FALSE

Utilzando SAMPLE:

Agora vamos testar uma situação com a função acima.

sample(10)
 [1]  3  1  5  8  2  6 10  9  4  7
# Exxemplo: sample (conjuntoDeNumeros, vezesQueVamosSortear, replace = FALSE, prob = NULL)
# 1:6, conjunto de 1 até 6, size UM é sinal que jogarei uma vez, e a reposição TRUE é sinal que um lado não deixa de existir
sample (x = 1:6, size = 1, replace = TRUE)
[1] 5
sample (x = 1:100, size = 4, replace = TRUE)
[1] 89 37 34 89
# Agora uma simulacao de um dado:
dado <- sample (x = 1:6, size = 100000, replace = TRUE)
sum(dado == 3)

mean(dado == 3)
